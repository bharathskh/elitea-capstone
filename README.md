# EliteA Demo Application

## Overview
EliteA Demo Application is a sample Node.js REST API created to demonstrate the EliteA Automated Documentation Sync workflow.

## Features
- REST API using Express
- Health Check endpoint
- Docker support
- GitHub Actions CI
- Jest unit testing

## Tech_Stack
- Node.js 20
- Express
- Jest

## Entry_Points
src/index.js

## Dependencies
npm install
